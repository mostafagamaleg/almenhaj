# Alminhaj Components

Reusable UI is centralized in `js/app.js` and styled by `css/style.css`.

Core components:

- App shell sidebar and topbar
- Stat cards
- Course cards
- Lesson rows
- Activity timeline
- Leaderboard rows
- Progress bars and rings
- Tables, badges, buttons, modals, toasts

All pages use semantic HTML placeholders and initialize shared components with `Alminhaj.initPage()`.

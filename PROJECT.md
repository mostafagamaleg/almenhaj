# مشروع المليون متدبّر — منصة المنهاج (Alminhaj LMS)

منصّة تعليمية إسلامية تفاعلية لتعلّم القرآن الكريم **تفسيراً وتدبّراً وتطبيقاً**، مع رحلة طالب متدرّجة، اختبارات فورية، لوحات تحكّم، شهادات، ومتصدّرين على مستوى الجمهورية.

> **الحالة:** نموذج واجهة أمامية (Front-end Prototype) تفاعلي وكامل بصرياً — **بدون باك-إند بعد**.
> **آخر تحديث:** 2026-06-26

---

## 1. نظرة عامة

الفكرة الأساسية أن يمرّ الطالب برحلة من **ست خطوات** لكل سورة:

1. الدخول لمسار «تفسير جزء عمّ» من الرحلة القرآنية.
2. اختيار السورة المراد تفسيرها.
3. مشاهدة فيديو شرح الشيخ.
4. قراءة التفسير **آية بجانب تفسيرها** مع حفظ الملاحظات.
5. اختبار تفاعلي على السورة بمؤقّت وتصحيح فوري.
6. عرض النتيجة ومتابعة الدرس التالي أو استلام الشهادة.

## 2. التقنية المستخدمة

| العنصر | التفاصيل |
|--------|----------|
| البنية | HTML ثابت متعدّد الصفحات (Multi-page) — يعمل مباشرة من المتصفح |
| التنسيق | Tailwind CSS عبر CDN + ملف `css/style.css` (Design System كامل) |
| المنطق | JavaScript خام (Vanilla) — `js/data.js`, `js/app.js`, `js/navigation.js` |
| الأيقونات | Lucide Icons |
| الخطوط | Cairo (واجهة) · Amiri (نص قرآني) · Inter (إنجليزي) |
| التخزين | `localStorage` لحفظ نتيجة الاختبار |
| اللغة | عربي RTL مع تبديل تجريبي إلى الإنجليزية LTR |

**لا يوجد على جهاز التطوير حالياً:** Node.js / Python / PowerPoint / LibreOffice — لذلك التطبيق يُفتح كملفات `file://` مباشرة.

### نمط الصفحات
كل صفحة داخلية تحتوي على:
- `<div class="alm-shell" data-layout="app"></div>` — حاوية الـ Shell.
- `<template data-page-content>...</template>` — محتوى الصفحة.
- استدعاء `Alminhaj.initPage({ active, title, subtitle })` الذي يبني الشريط الجانبي والشريط العلوي ويحقن المحتوى ويشغّل كل المحرّكات التفاعلية.

## 4. المزايا التفاعلية المنفّذة

| الميزة | الملف | الوصف |
|--------|-------|-------|
| **محرّك الاختبار** | `quiz.html` + `app.js:initQuiz` | 10 أسئلة من البيانات، اختيار بالضغط، مؤقّت تنازلي حقيقي، تنقّل سابق/تالي، حساب الدرجة وتخزينها |
| **النتيجة الديناميكية** | `result.html` + `app.js:renderResult` | تقرأ الدرجة الفعلية من `localStorage` وتعرض صحيح/خطأ/النسبة/الوقت وتغيّر الرسالة حسب النجاح |
| **التفسير آية بآية** | `tafsir-reader.html` + `app.js:initTafsir` | قائمة آيات سورة النبأ، الضغط على آية يعرض نصّها وتفسيرها + الآية التالية |
| **الخريطة الذهنية** | `mindmap.html` + `app.js:renderMindmap` | محاور سورة النبأ الأربعة من البيانات |
| **فلترة المكتبة** | `course.html` + `app.js:initCourseFilter` | فلترة المسارات حسب التصنيف |
| **رافع الملفات/الفيديو** | `upload.html` + `app.js:initUploader` | سحب وإفلات، معاينة فيديو/صورة حقيقية، شريط تقدّم، تحقّق من النوع والحجم، حذف، نشر |
| **نظام المحاضرات الديناميكي** | `lectures.html` + `app.js:initLecturesPage` | تصفّح بالفئات (7 فئات)، كروت المحاضرات مع حالة النشر والمصدر |
| **رفع محاضرة متكامل** | `upload.html` + `app.js:initLectureUpload` | يوتيوب (معاينة iframe فورية) أو ملف محلي، سلسلة أو مستقلّة، مرفقات منفصلة |
| الشريط الجانبي والتنقّل | `navigation.js` | فتح/غلق على الموبايل، تبديل اللغة، Toast، Modal |

> **ملاحظة عن الرفع:** الرفع حالياً من جهة المتصفح فقط (معاينة + محاكاة تقدّم). الملفات **لا تُحفظ على سيرفر** لعدم وجود باك-إند.

## 5. كيفية التشغيل

نظراً لعدم وجود Node/Python، شغّل التطبيق مباشرة:

- **افتح** `index.html` بنقرة مزدوجة، أو الصق في المتصفح:
  ```
  file:///D:/Almenhaj/index.html
  ```
- العرض التقديمي: `file:///D:/Almenhaj/presentation.html` (مفاتيح: `←/→` تنقّل، `F` ملء الشاشة، `P` طباعة PDF).

> لتشغيل سيرفر حقيقي بـ `http://localhost` (يحلّ قيود `localStorage` على `file://`) يلزم تثبيت Node.js.
